Basic Borrower Example

Install this example (terminal):


    npm i

Create this example from scratch:


    npm init
    npm i borrower --save
    ./node_modules/.bin/borrower install highlightjs --save
    edit index.html
    # put-in the source of this page

Clean:


    rm -rf node_modules
    rm -rf static_modules

example:


    define(['highlightjs'], function (hljs) {
        hljs.initHighlighting();
    });
